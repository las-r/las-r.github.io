binariter

binariter is a programming ide made to write code in pure binary glory.

it’ll have support for syntax highlighting and themes, similar to vscode.

basically, you input your binary code and you get the output.

you’ll have the option to write in pure binary or write in a new programming language called biniri.

biniri

biniri is a programming language made to write pure binary glory without the hassle.

biniri will use a register system for variables. here is an example with 8 registers and 4 bits per register, 32 bits of storage total:

register id register value
0 0000
1 0010
10 0101
11 0110
100 0100
101 1000
110 1010
111 1001
